# Home page
ここは、[おかゆグループ](https://okayugroup.com)のホームページです。

Next.js + Tailwind CSS で作成されています。

## 開発

```bash
git clone https://github.com/okayugroup/homepage.git
cd homepage
npm install
npm run dev
```

## デプロイ

```bash
npm run build
```

また、Cloudflare Pagesと連携して、GitHubにpushした際に自動でデプロイされるようになっています。
